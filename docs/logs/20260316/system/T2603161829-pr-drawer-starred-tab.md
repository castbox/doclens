# 任务简报：PR 文件面板星标文档 Tab

## 变更概述

- 在右侧 PR 文件面板新增 `Tab` 视图切换：
  - `全部文件`
  - `星标文档`
- `星标文档` 视图直接复用现有 PR 文件数据中的星标状态，不新增接口。
- 星标视图按最近星标时间倒序展示，便于快速回看重点文件。

## 影响文件

- `src/features/reviews/ui/ReviewDrawer.tsx`

## 行为说明

- 保持原有 PR 文件面板的类别筛选、已读状态筛选、点击打开文件、自动标记已读、列表内加星/取消星行为不变。
- 切换到 `星标文档` Tab 后，仅展示当前 PR 文件集合中已加星的文件。
- 当存在类别/已读筛选时，星标视图与全部文件视图都会沿用相同筛选条件。
- 星标视图为空时，按“暂无任何星标”与“当前筛选无结果”两种场景分别给出提示。

## 验证记录

- 构建校验：`pnpm build`
- 结果：通过

## 风险与后续建议

- 当前 `星标文档` Tab 聚焦于 `pr/` 目录下文件，与审阅列表页的全局星标文档汇总范围不同；如果后续希望右侧面板也查看全量星标文档，需要再明确产品口径并补充独立数据源。
