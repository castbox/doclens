# 任务简报：Docs 目录抽屉星标文档 Tab

## 变更概述

- 为左侧 `Docs` 抽屉新增 `目录树 / 星标文档` 两个 Tab，可在目录浏览与重点文档回看之间快速切换。
- 新增独立的星标列表组件，直接复用 `/api/docs/stars` 现有接口，不新增后端能力。
- 星标文档视图支持：
  - 按最近星标时间倒序显示
  - 按关键词筛选文件名和路径
  - 沿用当前 `scope` 目录范围过滤
  - 在列表内直接取消星标并打开文档

## 影响文件

- `src/features/docs/ui/DocsWorkspace.tsx`
- `src/features/docs/ui/DocsStarredList.tsx`

## 行为说明

- 保持原有目录树浏览、按路径打开文档、移动端选中文档后自动收起抽屉等行为不变。
- 当文档详情页或右侧 PR 文件面板更新星标状态后，左侧星标文档 Tab 会同步刷新。
- 当前如果设置了 `scope`，星标文档列表只展示该目录范围内的星标文件，确保与目录树视图语义一致。
- 从左侧星标列表取消星标时，仅更新星标状态，不影响文档只读与路径安全约束。

## 验证记录

- 构建校验：`pnpm build`
- 结果：通过

## 风险与后续建议

- 左侧抽屉标题仍沿用 `Docs 目录`，但内容已扩展为目录树和星标文档双视图；若后续继续增加最近访问等功能，可以统一升级为更泛化的 `Docs 面板` 命名。
