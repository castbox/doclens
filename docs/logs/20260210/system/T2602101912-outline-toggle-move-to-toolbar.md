# 大纲开关按钮位置调整简报

## 1. 变更目标
- 将“文档大纲展开/收起”按钮移动到详情页顶部工具栏，并放在“打开原文件”按钮右侧。
- 保持“收起后给正文更大展示空间”的行为。

## 2. 主要改动
### 2.1 按钮位置迁移
- 文件：`src/features/docs/ui/DocPreview.tsx`
- 在顶部操作按钮组中新增大纲开关按钮，位置紧随“打开原文件”按钮。
- 仅在 Markdown 且存在大纲时显示按钮。

### 2.2 折叠行为
- 文件：`src/features/docs/ui/DocPreview.tsx`
- 收起时不再渲染右侧大纲栏，正文区域直接获得更多水平空间。
- 展开时恢复右侧大纲栏渲染。

### 2.3 DocOutline 组件回归纯展示
- 文件：`src/features/docs/ui/DocOutline.tsx`
- 移除组件内的折叠开关与受控/非受控状态逻辑。
- 保留大纲列表展示与点击跳转能力。
